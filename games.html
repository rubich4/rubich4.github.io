ASTEROIDS ++

Asteroids ++ is a modernized version of the classic game Asteroids, it contains new features such user managment, local multiplayer, best scores, cooperative, singleplayer, saved games, among any other features.The objective of the game is to destroy all the asteroids that apper on the screen, acquiring scores and beating levels.

Features:

  • User Managment System
  • Save Game System
  • Cooperative Multiplayer Modes
  • Admin Mode
  • Best Highscores

Multiplayer cooperative mode & versus pvp mode available!

THIS IS NOT JETPAC

This is not Jetpac, it's a 2D platform shooter that takes place on a remote planet from the solar system. Hungry hostile creatures inhabit the plant so you will have to repair and refuel the spaceship to get out as quickly as possible. Play as an astronaut and defeat all the bosses and enemies that get in your way.

This game features challenging difficulty throughout the levels, fight against the Destruction Machine and the Giant Super Robot!

  • Destruction Machine -> Phase 1 Boss
  • Giant Super Robot -> Phase 2 Boss

Multiplayer cooperative mode available!
